    变量和简单的数据类型

    变量

    explem:

    1. 变量的命名：
      • 变量名由字母、下划线、数字组成，数字不能开头
      • 不要把python关键字和函数名作为变量名
      • 慎用小写字母l和o（易与1、0混淆）
    2. 变量是标签
    字符串
    1. 字符串可以是单引号也可以是多引号
    2. 修改大小写
      • ```
        name = ‘This is a string’
        print(name.title()) # 让python对name执行title（）操作，title（）第一个字符大写，其余小写
      • 同理，name.upper()
      • name.lower()
    3. 字符串中插入变量 这种字符串名叫f字符串，f在前引号前面，在python3.6版本引入，在此之前的版本用format（）
    4. 添加空白
      ‘/n’:换行
      ‘/t’:制表
    5. 删除空白
      • 删除开头的空白name.lstrip()
      • 删除结尾的空白name.rstrip()
      • 删除两边的空白name.strip()
      • 上述删除是暂时的，永久删除需要
    数
    1. 整数
      • 可以正常加减乘除运算
      • 乘方运算“x**y”
    2. 浮点数
    3. 整数和浮点数
      • 两数相除，结果总是浮点数
      • 有一个操作数是浮点数，结果也是浮点数。
    4. 下划线（3.6版本及以上支持）
      用来表示很大的数字，python会自动忽略’_’
    5. 多变量赋值
    6. 常量
      全大写表示常量

    ‘#’为python的注释符
